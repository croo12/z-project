# 2025-12-30 TODO: Run on Android Device

## Goal
Deploy and run the Rust/Tauri app on a physical Android device.

## 1. Prerequisites (Morning)
- [ ] **Install Android Studio**: [Download Link](https://developer.android.com/studio).
- [ ] **Configure SDK Manager**:
    - Open Android Studio -> More Actions -> SDK Manager.
    - **SDK Platforms**: Install Android API 34 (or latest stable).
    - **SDK Tools**: Check and install:
        - Android SDK Build-Tools
        - NDK (Side by side)
        - Android SDK Command-line Tools (latest)
        - Google USB Driver (for Windows)
- [ ] **Verify Environment Variables**:
    - `ANDROID_HOME`: `C:\Users\jimmy\AppData\Local\Android\Sdk`
    - `NDK_HOME`: `C:\Users\jimmy\AppData\Local\Android\Sdk\ndk\<version>`

## 2. Deployment (Afternoon)
1.  **Prepare Phone**:
    - Go to **Settings > About Phone**. Tap **Build Number** 7 times to enable Developer Options.
    - Go to **Developer Options** and enable **USB Debugging**.
    - Connect to PC via USB.
2.  **Initialize & Run**:
    ```bash
    # Initialize Android Project (if you haven't yet)
    npm run tauri android init
    
    # Run on Device
    npm run tauri android dev
    ```
3.  **Verification**:
    - App should launch on your phone.
    - Check "News" and "Add Todo" features work.
